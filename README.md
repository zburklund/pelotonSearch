# pelotonSearch

A React single-page application for searching Peloton on-demand classes using the unofficial Peloton API.

## Features
- Login with your Peloton credentials
- Browse and filter classes by category
- Search classes by title
- View class details: thumbnail, instructor, duration, difficulty

## Setup

```bash
npm install
npm run dev
```

App runs at `http://localhost:5173`
